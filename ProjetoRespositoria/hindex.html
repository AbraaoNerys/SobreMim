🛣️ Roadmap para Desenvolvedor Java Backend

1. Fundamentos da Programação

  • ✅ Lógica de programação
  • ✅ Estruturas de dados (listas, filas, pilhas, mapas)
  • ✅ Algoritmos básicos
  • ✅ Orientação a Objetos (POO)

2. Fundamentos do Java

  • ✅ Sintaxe da linguagem Java
  • ✅ Tipos de dados, variáveis, operadores
  • ✅ Controle de fluxo (if, switch, loops)
  • ✅ Métodos e escopos
  • ✅ POO com Java: classes, herança, polimorfismo, encapsulamento
  • ✅ Manipulação de exceções

6. Banco de Dados

  • 🔄 SQL com MySQL ou PostgreSQL
  • 🔄 NoSQL com MongoDB (opcional)

  • 🔄 JPA / Hibernate

7. Testes

  • 🔄 Mockito
  • 🔄 Testes de integração com Spring

8. DevOps (Intermediário)

9. Projetos Pessoais

  • 📌 API REST com Spring Boot
  • 📌 Integração com banco de dados relacional
  • 📌 Autenticação com Spring Security
  • 📌 Deploy em ambiente de produção
